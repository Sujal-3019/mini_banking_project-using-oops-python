# 🏦 Mini Banking Management System

> A simple banking application built with **Python** and **Streamlit** as a practice project to strengthen Object-Oriented Programming (OOP) concepts and file handling in Python.

---

## 📖 Overview

The **Mini Banking Management System** is a beginner-friendly banking application that simulates basic banking operations through an interactive web interface.

This project was developed as a **revision project** to reinforce Python fundamentals, especially **Object-Oriented Programming (OOP)**, while also practicing JSON-based data storage and building web applications with Streamlit.

---

## ✨ Features

- 👤 Create a new bank account
- 💰 Deposit money securely
- 💸 Withdraw money
- 📊 Check account balance
- 🔒 PIN-based authentication
- 📁 JSON database for persistent storage
- 📈 Dashboard showing:
  - Total Accounts
  - Total Bank Balance
- ✅ Input validation for:
  - Age
  - Email
  - PIN
  - Duplicate Account Numbers

---

## 🚀 Getting Started

### 1. Clone the Repository

```bash
git clone https://github.com/your-username/mini-banking-management-system.git
cd mini-banking-management-system
```

### 2. Install Dependencies

```bash
pip install -r requirements.txt
```

### 3. Run the Application

```bash
streamlit run app.py
```
